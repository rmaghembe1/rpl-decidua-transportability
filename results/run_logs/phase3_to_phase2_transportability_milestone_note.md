## Pilot group effect in GSE161969

- Controls: n = 3
- RPL: n = 4
- Mean score in controls: -0.582
- Mean score in RPL: 0.437
- Delta mean RPL minus control: +1.019
- Wilcoxon p-value: 0.114
- BH FDR: 0.457

## Interpretation

This result supports a positive directional transportability signal for a recurrent-miscarriage decidual inflammatory macrophage/APC programme across a single-cell-derived disease reference and an independent bulk decidua cohort.

This should be interpreted as exploratory and directional only. The GSE161969 pilot dataset is small, with n=3 controls and n=4 RPL samples, so the effect should not be described as statistically validated or as a standalone biomarker finding.
