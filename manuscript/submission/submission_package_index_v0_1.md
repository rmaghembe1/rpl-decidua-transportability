# Submission package index v0.1

## Project

Single-cell-informed transcriptomic transportability of decidual inflammatory macrophage/APC programmes in recurrent pregnancy loss

## Repository

https://github.com/rmaghembe1/rpl-decidua-transportability

## Current manuscript package status

This repository contains a reproducible computational manuscript package centred on a GSE214607-derived recurrent-miscarriage decidual inflammatory macrophage/APC programme that shows concordant positive directional recurrence across three independent bulk decidua cohorts: GSE161969, GSE113790 and GSE178535.

The current manuscript draft is exploratory and pre-submission. It is suitable for continued editing, reference insertion, journal formatting and final figure/table preparation.
