# Submission statements v0.1

## Data availability statement

This study reanalysed publicly available transcriptomic datasets. The single-cell reference dataset used for programme derivation was GSE214607. External bulk decidua transcriptomic cohorts used for transportability testing were GSE161969, GSE113790 and GSE178535. These datasets are available through the Gene Expression Omnibus.

Derived compact signatures, curated metadata, projection scores, transportability summaries, figure-support files, analysis scripts and release documentation are available in the project GitHub repository:

https://github.com/rmaghembe1/rpl-decidua-transportability

Large raw public files and large regenerated intermediate single-cell objects are not included in the GitHub repository but can be regenerated from the public source datasets using the documented scripts and release manifest.

## Code availability statement

All code used for dataset auditing, import, quality control, single-cell/pseudobulk programme derivation, signature projection, transportability summarisation and figure generation is available in the GitHub repository:

https://github.com/rmaghembe1/rpl-decidua-transportability

The repository includes scripts, compact derived signatures, curated metadata, transportability outputs, figures, release manifests and reproducibility notes.

## Ethics statement

This study was a secondary computational reanalysis of publicly available, de-identified transcriptomic datasets. No new human participants were recruited, no new human biological samples were collected and no identifiable participant-level data were analysed. Ethical approval and informed consent for the original studies were the responsibility of the original data generators and are described in the source dataset records and associated publications where available.
