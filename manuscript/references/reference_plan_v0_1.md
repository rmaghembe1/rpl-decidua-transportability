# Reference plan v0.1

## Manuscript

Single-cell-informed transcriptomic transportability of decidual inflammatory macrophage/APC programmes in recurrent pregnancy loss

## Purpose

This file defines the citation strategy for the manuscript before formal reference insertion. The goal is to support the manuscript with a focused, evidence-appropriate bibliography while avoiding citation inflation, weakly related references or overclaiming.

## Citation principles

References should support one of the following roles:

1. recurrent pregnancy loss definition, burden and clinical heterogeneity;
2. decidual and maternal-fetal-interface immunology;
3. macrophage, antigen-presentation and inflammatory tissue-state biology in decidua;
4. single-cell RNA-seq and pseudobulk logic in reproductive tissues;
5. transcriptomic reproducibility, transportability and cross-cohort projection;
6. public-data reuse and reproducible computational biology;
7. the specific public datasets used in the analysis.

Avoid references that only support generic inflammation, generic macrophage biology or generic biomarker discovery unless directly needed for interpretation.

## Required citation blocks

### Block 1: Recurrent pregnancy loss background

Use a compact set of authoritative clinical/review references to support:

- recurrent pregnancy loss as a clinically heterogeneous syndrome;
- variation in definitions and work-up;
- limitations of single-mechanism explanations;
- need for molecularly interpretable subgroup/tissue-state analyses.

Target number: 3-5 references.

### Block 2: Decidual immune biology and maternal-fetal interface

Use references supporting:

- decidua as an immune-active tissue;
- maternal immune tolerance, tissue remodelling and implantation biology;
- decidual immune-cell diversity;
- macrophages and antigen-presenting cells as context-dependent homeostatic and inflammatory populations.

Target number: 4-7 references.

### Block 3: Single-cell reproductive tissue atlases

Use references supporting:

- single-cell RNA-seq characterization of decidua, placenta or maternal-fetal-interface tissues;
- cell-type-informed interpretation of bulk transcriptomic data;
- pseudobulk or cell-state programme derivation.

Target number: 3-6 references.

### Block 4: Public datasets used in this manuscript

Cite original dataset papers or GEO records for:

- GSE214607;
- GSE161969;
- GSE113790;
- GSE178535.

Target number: all dataset source records and associated papers when available.

### Block 5: Transportability and reproducible transcriptomics

Use references supporting:

- cross-cohort validation/transportability problems in transcriptomics;
- limitations of DEG-only biomarker discovery;
- projection/scoring of gene programmes across datasets;
- reproducible workflows, public code and data transparency.

Target number: 4-8 references.

### Block 6: Evidence boundaries

Use references, if needed, to justify:

- exploratory public-data reanalysis;
- why transcriptomic recurrence does not equal causality;
- why diagnostic claims require dedicated validation.

Target number: 2-4 references.

## Citation placement plan

### Introduction

Needs citations for:

- RPL background and heterogeneity;
- decidual immune biology;
- limitations of isolated DEG lists;
- value of single-cell-informed programme projection.

### Methods

Needs citations for:

- datasets/GEO records;
- pseudobulk and gene-programme scoring logic if using methodological sources;
- reproducibility or public-data standards if appropriate.

### Results

Use few citations. Mostly cite dataset sources where first mentioned.

### Discussion

Needs citations for:

- macrophage/APC biology in decidua;
- interpretation of inflammatory antigen-presentation programmes;
- boundaries of causal/diagnostic inference;
- future validation directions.

## References to avoid overusing

Do not overload the manuscript with:

- generic macrophage polarization references unrelated to decidua;
- generic M1/M2 framing without decidual context;
- broad inflammation reviews unrelated to pregnancy loss;
- cancer or infection macrophage references unless explicitly used as methodological analogy;
- biomarker-discovery references that make the manuscript sound like a classifier paper.

## Reference completion workflow

1. Identify dataset source papers and GEO records.
2. Identify authoritative RPL clinical/background references.
3. Identify maternal-fetal-interface single-cell atlas references.
4. Identify decidual macrophage/APC biology references.
5. Identify transcriptomic transportability/reproducibility references.
6. Insert references into manuscript v0.4.
7. Generate a clean reference list in the format required by the selected journal.
8. Recheck every citation for claim support and avoid unsupported overstatements.
