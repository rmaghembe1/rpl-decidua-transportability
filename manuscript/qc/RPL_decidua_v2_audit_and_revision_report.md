# RPL decidua manuscript v2 audit and revision report

## Input audited

Uploaded/source manuscript: `RPL_decidua_scientific_reports_submission_v1_1.docx`.

## Summary of audit findings from v1.1

1. Figures and tables existed mainly as legends/end matter, but the main text did not consistently cite each figure/table at the point where the corresponding evidence was introduced.
2. Supplementary material captions were present, but Supplementary Tables/Figures were not consistently cited in the Materials and Methods or Results.
3. The reference list contained 12 items, adequate for a technical working draft but too sparse for a Scientific Reports manuscript intended to establish the clinical problem, maternal-fetal-interface biology, immune/cellular mechanisms and methodological rationale.
4. The Discussion was conservative and accurate, but it did not yet explicitly map each major result back to the Introduction's problem/gap/aims or compare each interpretive axis with published RPL, macrophage, immune-atlas and complement literature.

## Revisions implemented in v2

### Figures and tables

- Added explicit in-text citation of Fig. 1 and Table 1 in the Introduction/Methods/Results when the study design and dataset framework are introduced.
- Added explicit in-text citation of Table 2 and Fig. 2 where GSE214607-derived signatures are described.
- Added explicit in-text citation of Table 3 and Fig. 3 where the macrophage/APC three-cohort result is presented.
- Added explicit in-text citation of Fig. 4 where secondary-signature behaviour is interpreted.

### Supplementary material

- Added Supplementary Table S1 citation for metadata and sample inclusion.
- Added Supplementary Table S2 citation for full signature gene lists.
- Added Supplementary Table S3 citation for projection scores and coverage.
- Added Supplementary Table S4 citation for figure/reproducibility support.
- Added Supplementary Fig. S1/S2 citations for dataset processing and intermediate projection summaries.

### References

- Expanded the reference list from 12 to 40 references.
- Added clinical/guideline references, miscarriage-burden references, decidualization and embryo-selection references, maternal-fetal-interface single-cell atlases, immune crosstalk references, RPL immune-atlas studies, decidual macrophage papers, complement-related pregnancy papers, dataset/source references and method references.

### Introduction

- Rebuilt the Introduction around four layers: clinical problem, decidual/maternal-fetal-interface biology, RPL immune/cellular literature, and the public-data/methodological gap.
- Added explicit study aims.

### Discussion

- Reorganized the Discussion to interpret the results scientifically, biologically, methodologically and translationally.
- Added comparison with RPL immune-atlas studies, macrophage-focused RPL literature, complement/coagulation literature and methodological pseudobulk/transportability literature.
- Preserved claim boundaries: no causality, no diagnostic performance, no clinical utility, no biomarker validation.

## Current v2 status

The v2 manuscript is a revised working draft suitable for further local review and repository integration. It remains conservative and evidence-bounded.
